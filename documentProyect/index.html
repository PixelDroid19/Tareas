sus elementos se posiciona utilizando css grid para hacerlo reponsive para multiples dipositivos.

JavaScript se utilizo para crear un evento para el reproductor.

grid-template-columns: Define las columnas de un contenedor de cuadrícula.
Puede especificar el ancho de una columna usando una palabra clave (como auto).

css

/*define el número de columnas de la cuadrícula*/ grid-template-columns: repeat(auto-fit, minmax(150px, auto));

función repeat(): permite repetir columnas tantas veces como sea necesario.
Por ejemplo, si está creando una cuadrícula de 12 columnas, podría escribir la siguiente línea:

css

.grid { display: grid; /*define el número de columnas de la cuadrícula*/ grid-template-columns: repeat(12, 1fr); }

JavaScript

function toggleVideo(){ const trailer = document.querySelector('.trailer');//Selector para la clase trailer const video = document.querySelector('video');//Selector para la etiqueta trailer trailer.classList.toggle('active');//si es visible está o presente lo elimina, de lo contrario la añade. video.currentTime = 0; video.pause(); }

auto-fill: llena la fila con tantas columnas como pueda caber.
Por lo tanto, crea columnas implícitas siempre que una nueva columna puede caber, porque está tratando de llenar la fila con tantas columnas como sea posible.
Las columnas recién agregadas pueden estar vacías, pero seguirán ocupando un espacio designado en la fila.

minmax(): especifica el ancho y mínimo de las columna.

Calculator

La calculadora utiliza funciones nativas y no nativas(hecha por el programador)
de javascript para realizar las operaciones matematicas

funciones nativas utilizadas:

  • eval(): evalúa un código JavaScript representado como una cadena de caracteres.
  • setTimeout: nos permite ejecutar una función una vez después del intervalo de tiempo.

funciones no nativas utilizadas:

  • getData(): se encarga obtener de los input y luego mostrarla en pantalla
  • calculater(): utiliza eval() para realizar la operacion y luego mostrarla en pantalla

Algoritmo:

  1. Obtener valores
  2. evalua operacion matematica
  3. mostrar resultado
